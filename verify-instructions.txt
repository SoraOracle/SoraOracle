Why you only see bytecode:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

BSCScan only has your compiled bytecode (0x608060...).
You need to VERIFY your contract to show the source code!

Quick Fix (3 steps):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Get FREE API key:
   → Visit: https://bscscan.com/myapikey
   → Register and copy your API key

2. Add to .env:
   BSCSCAN_API_KEY=YourAPIKeyHere

3. Verify:
   npx hardhat verify --network bscTestnet \
     0xA215e1bE0a679a6F74239A590dC6842558954e1a \
     0x29ecD8FA7D7249e791B2563f83De4c124e639B90

After verification:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ Source code visible on BSCScan
✅ Green 'Verified' checkmark
✅ Read/Write tabs appear
✅ Users can interact directly

See VERIFY_CONTRACT.md for detailed guide!
